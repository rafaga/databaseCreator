# databaseCreator

Utility to dump SDE data into a ralational database

Supported Databases:

  * SQLite

The tool can be customized to import just what do you need, and discard everything else

What data it is exported

  * Solar Systems (Planets, Moons, Stars)
  * Regions
  * Constellations
  * NPC stations
  * Gates
